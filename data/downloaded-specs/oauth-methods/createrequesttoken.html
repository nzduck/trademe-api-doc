CreateRequestToken

Creates an oauth request token with the given consumer information. This is the first step in the OAuth 1 process

URL: https://api.trademe.co.nz/OAuth/RequestToken.{file_format}
HTTP Method: GET
Requires Authentication? Yes
Permission Required: None
Supported Formats: XML, JSON
Rate Limited? Yes

URL parameters

file_format Enumeration (required) The format of the response.

Returns

Nothing.
