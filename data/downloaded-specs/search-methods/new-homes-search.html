New homes search

Performs a search in New Homes of Trade Me Property

URL: https://api.trademe.co.nz/v1/Search/Property/NewHomes.{file_format}
HTTP Method: GET
Requires Authentication? Yes
Permission Required: None
Supported Formats: XML, JSON
Rate Limited? Yes
