Save a category to your favourites

Saves a category to the currently authenticated user's list of favourite categories.

URL: https://api.trademe.co.nz/v1/favourites/category.{file_format}
HTTP Method: POST
Requires Authentication? Yes
Permission Required: MyTradeMeWrite: Maintain your watchlist, answer questions and post comments.
Supported Formats: XML, JSON
Rate Limited? Yes

URL parameters

file_format Enumeration (required) The format of the response.

POST Data

<SaveCategoryRequest>

Email Enumeration (required)

The frequency that emails should be sent.
None 0

No emails are sent.
Daily 1

An email is sent every day.
Every3Days 3

An email is sent each third day.
Weekly 7

An email is sent every week.
Push Enumeration (optional)

The frequency that push notifications should be sent.
None 0

Push not enabled for this favourite.
Instant 1

Push is enabled for this favourite.
Label String (optional)

The label of the favourite should be sent.
CategoryId Integer (required)

The ID of the category to save. This must be a leaf category.

Returns

<SavedFavouriteResponse>

Details of the saved favourite, including the ID.

SearchId Integer

The ID of the saved favourite.
Response String or null

A description of the error, if the operation failed.
Saved Boolean

Indicates whether the favourite was saved.
FavouriteType Enumeration

Indicates what type of favourite was saved.
Label String or null

Display Label of favourite was saved.

Example JSON Request 

{
  "Email": 0,
  "Push": 0,
  "Label": "ABC",
  "CategoryId": 123
}

Example JSON Response 

{
  "SearchId": 123,
  "Response": "ABC",
  "Saved": false,
  "FavouriteType": 0,
  "Label": "ABC"
}
