Retrieve all Employment duration type values

Returns a list of Employment duration types

URL: https://api.trademe.co.nz/v1/Jobs/Catalogue/EmploymentDurationTypes.{file_format}
HTTP Method: GET
Requires Authentication? Yes
Permission Required: None
Supported Formats: XML, JSON
Rate Limited? Yes

URL parameters

file_format Enumeration (required) The format of the response.

Returns

Collection of <CatalogueItem>

Label String or null

The display label of the item
Value String or null

The value of the item

Example JSON Response 

[
  {
    "Label": "ABC",
    "Value": "ABC"
  },
  {
    "Label": "ABC",
    "Value": "ABC"
  }
]
