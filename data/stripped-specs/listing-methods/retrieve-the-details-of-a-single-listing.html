Retrieve the details of a single listing

Retrieves the details of a single listing.

URL: https://api.trademe.co.nz/v1/listings/{listingId}.{file_format}
HTTP Method: GET
Requires Authentication? Yes
Permission Required: None
Supported Formats: XML, JSON
Rate Limited? Yes

URL parameters

listingId Integer (required)

The ID of the listing to retrieve.
file_format Enumeration (required) The format of the response.

Query String parameters

increment_view_count Boolean (optional)

Indicates whether to increment the view count on the listing. Defaults to true.
question_limit Integer (optional)

The maximum number of questions and answers to return. Defaults to unlimited.
return_member_profile Boolean (optional)

If true also returns the member's public profile information. Defaults to false.
return_variants Boolean (optional)

If true, returns full details of all variants in the VariantDefinition field. Only applies to MVL listings.
rsqid String (optional)

The referring search query ID; used to link requests to searches for analytics purposes. Optional.
