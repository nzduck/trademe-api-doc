Retrieves complementary listings

Retrieves complementary items for a given listing.

URL: https://api.trademe.co.nz/v1/listings/{listingId}/Complementary.{file_format}
HTTP Method: GET
Requires Authentication? Yes
Permission Required: None
Supported Formats: XML, JSON
Rate Limited? Yes

URL parameters

listingId Integer (required)

The ID of the listing to retrieve complementary results of.
file_format Enumeration (required) The format of the response.

Query String parameters

rsqid String (optional)

The referring search query ID; used to link requests to searches for analytics purposes. Optional.

Returns

<ComplementaryListings>

Listings that are complementary to the listing with the given listing ID.

This field cannot cope with the two main geographical classification systems. Except for the property search API, it should not be used.
