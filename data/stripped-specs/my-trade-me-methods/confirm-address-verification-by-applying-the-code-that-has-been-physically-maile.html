Confirm address verification by applying the code that has been physically mailed to the member

URL: https://api.trademe.co.nz/v1/mytrademe/ApplyAddressVerificationCode.{file_format}
HTTP Method: POST
Requires Authentication? Yes
Permission Required: MyTradeMeWrite: Maintain your watchlist, answer questions and post comments.
Supported Formats: XML, JSON
Rate Limited? Yes

URL parameters

file_format Enumeration (required) The format of the response.

POST Data

<ApplyAddressVerificationCodeRequest>

AddressVerificationCode String (optional)

The verification code that has been physically mailed to the member

Returns

<ApplyAddressVerificationCodeResponse>

A response that indicates whether the operation succeeded.

Success Boolean

Indicates whether the operation was successful.
Description String or null

The description of the error, if the operation failed.
