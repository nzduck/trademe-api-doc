Remove the member's profile image

Removes the member's profile image

These images are retained indefinitely.

URL: https://api.trademe.co.nz/v1/photos/memberprofile.{file_format}
HTTP Method: DELETE
Requires Authentication? Yes
Permission Required: MyTradeMeWrite: Maintain your watchlist, answer questions and post comments.
Supported Formats: XML, JSON
Rate Limited? Yes

URL parameters

file_format Enumeration (required) The format of the response.

Returns

<PhotoResponse>

Success if the image was removed. Failure otherwise.

Status Enumeration

The status of the photo operation.

Example JSON Response 

{
  "Status": 0
}
