Clear recent searches

Clear any or all recent searches recorded against the user's profile.

URL: https://api.trademe.co.nz/v1/Search/RecentSearches.{file_format}
HTTP Method: DELETE
Requires Authentication? Yes
Permission Required: MyTradeMeWrite: Maintain your watchlist, answer questions and post comments.
Supported Formats: XML, JSON
Rate Limited? Yes

URL parameters

file_format Enumeration (required) The format of the response.

Query String parameters

search_term String (optional)

The search term to remove. If this is not supplied, then all search terms are removed from the user's profile.

Returns

<SearchSuggestionsResponse>

Details on whether the operation succeeded.

Success Boolean

Indicates whether the operation was successful.
Description String or null

The description of the error, if the operation failed.

Example JSON Response 

{
  "Success": false,
  "Description": "ABC"
}
