Modify search suggestion settings

Modifies the user's search suggestion settings.

URL: https://api.trademe.co.nz/v1/Search/Suggestions.{file_format}
HTTP Method: POST
Requires Authentication? Yes
Permission Required: MyTradeMeWrite: Maintain your watchlist, answer questions and post comments.
Supported Formats: XML, JSON
Rate Limited? Yes

URL parameters

file_format Enumeration (required) The format of the response.

POST Data

<SearchSuggestionSettingsRequest>

RecentSearchesEnabled Boolean (required)

Whether or not to enable recording of recent searches against the user's profile. Setting this to false clears the user's recent searches and prevents search queries from being recorded.

Returns

<SearchSuggestionsResponse>

Details on whether the operation succeeded.

Success Boolean

Indicates whether the operation was successful.
Description String or null

The description of the error, if the operation failed.

Example JSON Request 

{
  "RecentSearchesEnabled": false
}

Example JSON Response 

{
  "Success": false,
  "Description": "ABC"
}
