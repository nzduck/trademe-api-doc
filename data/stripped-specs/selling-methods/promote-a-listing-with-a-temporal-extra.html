Promote a listing with a temporal extra

Promote a listing with a Temporal (time based) Extra

URL: https://api.trademe.co.nz/v1/Selling/ListingTemporalExtraPromotion.{file_format}
HTTP Method: POST
Requires Authentication? Yes
Permission Required: MyTradeMeWrite: Maintain your watchlist, answer questions and post comments.
Supported Formats: XML, JSON
Rate Limited? Yes

URL parameters

file_format Enumeration (required) The format of the response.

POST Data

<ListingTemporalExtraPromotionRequest>

ListingId Integer (required)

The id of the listing to promote
Extra Enumeration (required)

The extra to promote the listing with
Boost 0

Boost a Listing so it appears higher in search results
RenewSuperFeature 1

Renew expiring Super Feature.

Returns

<ListingTemporalExtraPromotionResponse>

Success Boolean

Indicates whether the operation was successful.
Description String or null

The description of the error, if the operation failed.
ListingId Integer

The ID of the listing.

Example JSON Request 

{
  "ListingId": 123,
  "Extra": 0
}

Example JSON Response 

{
  "Success": false,
  "Description": "ABC",
  "ListingId": 123
}
