Remove leading bid

Removes the leading bid from an auction. To remove a bid, you must be the seller of the auction.

URL: https://api.trademe.co.nz/v1/Selling/RemoveLeadingBid.{file_format}
HTTP Method: POST
Requires Authentication? Yes
Permission Required: MyTradeMeWrite: Maintain your watchlist, answer questions and post comments.
Supported Formats: XML, JSON
Rate Limited? Yes

URL parameters

file_format Enumeration (required) The format of the response.

POST Data

<RemoveBidRequest>

ListingId Integer (required)

The ID of the auction to remove the bid from. You must be logged in as the seller of the auction.
BidAmount Number (required)

The value of the bid to remove, in New Zealand dollars.
BidderId Integer (required)

The ID of the member who placed the bid.

Returns

<RemoveBidResponse>

Indicates whether or not the operation succeeded.

Success Boolean

Indicates whether the operation was successful.
Description String or null

The description of the error, if the operation failed.

Example JSON Request 

{
  "ListingId": 123,
  "BidAmount": 123.0,
  "BidderId": 123
}

Example JSON Response 

{
  "Success": false,
  "Description": "ABC"
}
