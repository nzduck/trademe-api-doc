Retrieves jobs dynamic pages directory

Retrieves jobs dynamic pages directory.

URL: https://api.trademe.co.nz/v1/seo/jobs/dynamicpagesdirectory.{file_format}
HTTP Method: GET
Requires Authentication? Yes
Permission Required: None
Supported Formats: XML, JSON
Rate Limited? Yes

URL parameters

file_format Enumeration (required) The format of the response.

Query String parameters

page Integer (optional)

The page of results to return, defaults to 1.
rows Integer (optional)

The number of results to return, defaults to 5000.

Returns

<DynamicPagesDirectoryInfo>

Page Integer

Current page
Rows Integer

Number of rows
Total Integer

Total records of dynamic pages categories
DynamicPagesCategories Collection of <DynamicPagesCategoryInfo> or null

List of dynamic page categories
